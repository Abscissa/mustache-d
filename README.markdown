# Mustache for D

Mustache is a push-strategy (a.k.a logic-less) template engine.

# Features

* Variables
* Sections
** Lists
** Non-False Values
** Lambdas(half implementation)
** Inverted
* Comments
* Partials

# Usage

See example directory.
